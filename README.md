# 아수라장 (Asurajang) Minecraft Plugin

## 프로젝트 소개
아수라장은 Paper 기반의 소규모 PvP 미니게임 플러그인입니다. 랜덤 바이옴 전장에서 팀전 또는 개인전으로 싸우며, 전투 중 무작위로 얻는 증강(특수 능력)으로 전투를 유리하게 이끌어 나갑니다.

---

## 주요 기능

### 전장 시스템
- 게임 시작 시 서버를 탐색해 무작위 바이옴을 전장으로 선정합니다.
- 전장 중심에 월드 보더가 생성되어 좁은 구역 안에서 전투가 진행됩니다.
- 게임 중에는 시간이 항상 아침으로 고정됩니다.
- 제한 시간이 끝나거나 관리자가 `/mayhem stop`으로 강제 종료하면 게임이 끝납니다.

### 증강 시스템
- 전투 중 무작위로 증강을 획득해 효과를 얻습니다.
- 총 23종의 증강이 존재하며, `/mayhem list`로 전체 목록과 설명을 확인할 수 있습니다.
- `/mayhem status`로 자신이 보유한 증강을 확인할 수 있습니다.

### 보상 시스템
- 적을 처치하면 골드와 경험치를 얻으며, 경험치가 쌓이면 레벨업합니다.
- 어시스트한 플레이어도 처치 보상을 함께 나눠 받습니다.

| 상황 | 골드 |
|------|------|
| 일반 킬 | +50 |
| 퍼스트 블러드 | +75 (+25 보너스) |

**연속킬 (10초 이내 추가 킬)**

| 연속 | 레이블 | 추가 골드 |
|------|--------|-----------|
| 2연 | 더블 킬 | +5 |
| 3연 | 트리플 킬 | +10 |
| 4연 | 쿼드라 킬 | +15 |
| 5연 | 펜타 킬 | +20 |
| 6연+ | 전설적인 킬 | +25 |

### 사망 및 리스폰
- 사망해도 아이템·경험치 드롭 없이 인벤토리가 그대로 유지됩니다.
- 사망 위치에서 10초간 관전한 뒤 자동으로 리스폰됩니다.

## 게임 모드

### 팀전
- 무작위로 레드팀 / 블루팀으로 나뉘어 서로 반대편에서 시작합니다.
- 이름표와 접두사(`[레드]` / `[블루]`)로 같은 편을 구분합니다.

### 개인전
- 모두가 적이며, 전장 곳곳에 흩어져 시작합니다.

### 기지 모드 (옵션)
- 팀전·개인전에 추가로 적용할 수 있는 거점 점령 모드입니다.
- 양 팀 진영에 거점이 생성되고, 거점을 지키는 가디언을 쓰러뜨리면 거점이 파괴됩니다.
- 가디언은 여러 번의 목숨을 가지고 있어, 한 번 쓰러뜨려도 시간이 지나면 더 강해져서 부활합니다.
- 같은 편끼리는 서로 피해를 주지 않으며, 거점 모드에서는 죽으면 우리 팀 거점에서 부활합니다.

## 명령어

| 명령어 | 권한 | 설명 |
|--------|------|------|
| `/mayhem start` | `asurajang.admin` | 게임 모드 선택 GUI 열기 |
| `/mayhem stop` | `asurajang.admin` | 게임 강제 종료 |
| `/mayhem reload` | `asurajang.admin` | 설정 리로드 |
| `/mayhem list` | `asurajang.admin` | 전체 증강 목록 보기 |
| `/mayhem status` | 없음 | 내 증강 확인 |

별칭: `/아수라장`, `/증바람`
