# 아수라장 (Asurajang)

Paper 기반 소규모 PvP 미니게임 플러그인입니다.  
랜덤 바이옴 전장에서 팀전 또는 개인전으로 10분간 싸웁니다.

---

## 요구 사항

| 항목 | 버전 |
|------|------|
| Minecraft | 26.1.2 |
| Paper | 26.1.2+ |
| Java | 25+ |

---

## 설치

```
mvn clean package
```

빌드된 `.jar`을 서버의 `plugins/` 폴더에 넣고 재시작합니다.

---

## 명령어

| 명령어 | 권한 | 설명 |
|--------|------|------|
| `/mayhem start` | `asurajang.admin` | 게임 모드 선택 GUI 열기 |
| `/mayhem stop` | `asurajang.admin` | 게임 강제 종료 |
| `/mayhem reload` | `asurajang.admin` | 설정 리로드 |
| `/mayhem list` | `asurajang.admin` | 전체 증강 목록 보기 |
| `/mayhem status` | 없음 | 내 증강 확인 |
| `/mayhem debug aug` | `asurajang.admin` | 증강 선택 GUI 디버그 |
| `/mayhem debug proc` | `asurajang.admin` | 흑섬 확률 100% 토글 |

별칭: `/아수라장`

---

## 게임 흐름

1. `/mayhem start` → 3줄 인벤토리 GUI에서 **팀전** 또는 **개인전** 선택
2. 랜덤 바이옴 전장 탐색 후 5초 카운트다운
3. 게임 시작 — 전장 중심에 50×50 월드 보더 적용, 시간 아침 고정
4. 10분 타이머 종료 또는 `/mayhem stop` 시 게임 종료
5. 플레이어 인벤토리 자동 복구

---

## 게임 모드

### 팀전
- 플레이어를 무작위로 **레드팀 / 블루팀** 으로 분배
- 레드팀 → NW 모서리 스폰, 블루팀 → SE 모서리 스폰
- 이름 태그·접두사(`[레드]` / `[블루]`)로 팀 구분

### 개인전
- 전장 내 무작위 위치에 스폰

## 킬 시스템

### 처치 메시지
```
Attacker님이 Victim님을 처치했습니다
```

### 골드 보상 (킬러에게만 표시)

| 상황 | 골드 |
|------|------|
| 일반 킬 | +50 |
| 퍼스트 블러드 | +75 (+25 보너스) |

### 멀티킬 (10초 이내 연속 킬)

| 연속 | 레이블 | 추가 골드 |
|------|--------|-----------|
| 2연 | 더블 킬 | +5 |
| 3연 | 트리플 킬 | +10 |
| 4연 | 쿼드라 킬 | +15 |
| 5연 | 펜타 킬 | +20 |
| 6연+ | 전설적인 킬 | +25 |

골드 획득 시 사슬 부숴지는 소리(`BLOCK_CHAIN_BREAK`, 피치 0.8) 재생

### 사망
- 아이템·경험치 드롭 없음, 인벤토리 유지
- 사망 위치에서 10초 관전 후 자동 리스폰

## 증강 시스템 (핫바 8번 슬롯 `증강 선택`)

게임 중 증강을 선택해 특수 효과를 얻습니다. 슬롯당 최대 3회 리롤 가능.

| 증강 | 효과 |
|------|------|
| 흑섬 | 공격 시 확률로 추가 피해 |
| 데드풀 | 체력이 낮을 때 자동 회복 |
| 부기우기 | 피격 시 공격자와 위치 교환 |
