# 아수라장 (mayhem)

Paper 기반 소규모 PvP 미니게임 플러그인입니다.  
랜덤 바이옴 전장에서 팀전 또는 개인전으로 싸웁니다.

---

## 요구 사항

| 항목 | 버전 |
|------|------|
| Minecraft | 26.1.2 |
| Paper | 26.1.2+ |
| Java | 25+ |

## 게임 흐름

1. `/mayhem start` → 3줄 인벤토리 GUI에서 **팀전 / 개인전** 선택 및 **기지 모드** 토글
2. 랜덤 바이옴 전장 탐색 → 슬롯머신 연출로 바이옴 공개 → 3초 카운트다운
3. 게임 시작 — 전장 중심에 96×96(6×6 청크) 월드 보더 적용, 시간 아침(1000틱) 고정, 제한시간 15분
4. 타이머 종료 또는 `/mayhem stop` 시 게임 종료

---

## 명령어

| 명령어 | 권한 | 설명 |
|--------|------|------|
| `/mayhem start` | `asurajang.admin` | 게임 모드 선택 GUI 열기 |
| `/mayhem stop` | `asurajang.admin` | 게임 강제 종료 |
| `/mayhem reload` | `asurajang.admin` | 설정 리로드 |
| `/mayhem list` | `asurajang.admin` | 전체 증강 목록 보기 |
| `/mayhem status` | 없음 | 내 증강 확인 |

별칭: `/아수라장`, `/증바람`

## 게임 모드

### 팀전
- 플레이어를 무작위로 **레드팀 / 블루팀** 으로 분배
- 레드팀 → NW 모서리 스폰, 블루팀 → SE 모서리 스폰
- 이름 태그·접두사(`[레드]` / `[블루]`)로 팀 구분

### 개인전
- 전장 내 무작위 위치에 스폰

### 기지 모드 (토글 옵션)
- 팀전·개인전에 추가로 적용 가능한 거점 점령전 모드
- 각 팀 모서리에 콘크리트 거점 플랫폼과 비콘을 생성하고, 그 위에 **거점 가디언**(보스바 체력 표시) 배치
- 가디언은 라이프제(200 → 300 → 400)로, 한 라이프가 깎이면 20초 회복 시간을 가지며 마지막 라이프가 파괴되면 거점 파괴 메시지를 전체 공지
- 같은 팀 피해는 무시되며, 거점 모드에서는 리스폰 위치도 자기 팀 플랫폼으로 고정

## 보상

### 골드

| 상황 | 골드 |
|------|------|
| 일반 킬 | +50 |
| 퍼스트 블러드 | +75 (+25 보너스) |
| 어시스트 (10초 이내 피해 기여) | 처치 보상을 어시스트 인원과 분배 (킬러는 절반, 나머지는 어시스트 인원 수만큼 균등 분배) |

### 연속킬 (10초 이내 추가 킬)

| 연속 | 레이블 | 추가 골드 |
|------|--------|-----------|
| 2연 | 더블 킬 | +5 |
| 3연 | 트리플 킬 | +10 |
| 4연 | 쿼드라 킬 | +15 |
| 5연 | 펜타 킬 | +20 |
| 6연+ | 전설적인 킬 | +25 |

### 경험치
- 처치 시 `킬러 레벨 × 10 + 피해자 킬 수 × 2 + 75` 만큼 획득, 누적 시 레벨업

---

## 증강

- 전투 중 획득해 효과를 얻는 패시브/액티브 능력으로, 총 23종 보유
- `/mayhem list`로 전체 목록과 설명 확인, `/mayhem status`로 자신이 보유한 증강 확인
- 게임 중 핫바의 "증강 선택" 아이템으로 무작위 증강 중 하나를 선택해 획득

- 아이템·경험치 드롭 없음, 인벤토리 유지
